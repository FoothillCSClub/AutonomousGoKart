## Dependencies:

* numpy
* Cython
* freenect
* vispy (for visualization of data)
* PyQt5 (for visualization: used by VisPy)

## Folders & Packages

* kart/: Package holding source code for project (analogous to 'src' 
    folder in Java or C)
    
* lib/: Directory containing compiled C for use by project.
